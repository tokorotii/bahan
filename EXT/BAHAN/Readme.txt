* Cara Jalanin Program *
1. Masukin folder program ke htdocs
2. Buka phpmyadmin trus bikin database baru namanya jewepe
3. Trus import databasenya kesitu di menu import
4. Buka VS Code trus buka folder jewepe yg di htdocs (biar kodingannya nampil :v)

5. Kalo mau ubah warna navbarnya ada di direktori template/header.php
   disitu ada style nav, nah background-color nya diganti
6. Kalo mau ubah tampilan home ada di direktori template/dashboard.php
   disitu bisa ubah tulisan judul ama gambarnya
7. Kalo mau ubah tampilan login ada di file login.php
   disitu ada style ubah aja seperlunya

8. Kalo mau login admin:
   username = admin
   password = admin

9. Kalo mau login mahasiswa lu tambahin dulu mahasiswa di admin
   kalo udah buat loginnya pake npm mahasiswa yg lu bikin tadi, contoh:
   username = 50418093
   password = 50418093


* Rancangan Database *
Buat rancangan databasenya, lu ss aja yg di phpmyadmin.
Caranya:
1. Buka database jewepe trus klik menu design atau desainer
2. Nah itu lu rapihin trus tinggal ss deh
3. Atau cara kedua lu bikin tabel manual di word isinya ngikutin yg disitu
